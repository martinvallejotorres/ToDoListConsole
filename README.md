# ToDo List Console - C#

Simple ToDo List application built with C# and .NET.

## Features

- Add tasks
- Modify task status
- Delete tasks
- View tasks
- JSON persistence
- LINQ usage

## Technologies

- C#
- .NET
- LINQ
- System.Text.Json

dotnet run
